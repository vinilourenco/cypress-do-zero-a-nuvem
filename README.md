# Cypress, from Zero to the Cloud

Sample project for the "Cypress, from Zero to Cloud" course of the Talking About Testing online school.

## Pre-requirements

It is required to have git, Node.js and npm installed to clone and run this project.

> I've used versions **2.53.0**, **v22.22.2** and **10.9.7** of git, Node.js and npm, respectively. I suggest you use the same or later LTS versions.
